# 二创风格详细说明

## 风格 1：🔥 爆款标题党

**适用场景**：流量导向内容、泛娱乐、热门话题
**核心原则**：用冲击力强的标题勾住读者，内容简短有力，结尾引导互动

### 写作规则
1. 标题必须包含数字（「3个」「90%」「月薪5000到5万」）
2. 制造信息差（「99%的人不知道」「这件事你一定做错了」）
3. 正文分点列举，每点一行，使用 emoji 分隔
4. 结尾必须有互动引导（「你试过吗？评论区聊聊」）
5. 全文控制在 300 字以内

### 标题模板
- `「数字」个让你XXX的方法，第X个太绝了`
- `我靠这个XXX，省了XXX，后悔没早知道`
- `别再XXX了！正确做法是这样的↓`

### 示例
```
# 5个让你月薪翻倍的副业，第3个躺着就能赚！

🔥 很多人不知道，其实这些副业门槛超低：

1️⃣ **XX方向**——每天1小时，月入3000+
2️⃣ **XX技能**——零基础可学，平台自带流量
3️⃣ **XX模式**——真的躺赚，我亲测有效
4️⃣ **XX渠道**——冷门但暴利，竞争少
5️⃣ **XX方法**——时间自由，适合上班族

💬 你在做哪个？评论区告诉我，一起交流！

#副业 #赚钱方法 #理财
```

## 风格 2：📖 深度干货流

**适用场景**：知识类、教程类、专业领域内容
**核心原则**：结构清晰、内容扎实、有深度，建立专业信任感

### 写作规则
1. 开头用「背景/痛点」引入，不超过 50 字
2. 使用标准 Markdown 标题层级（## / ###）
3. 每个要点都要有「原因 + 方法 + 案例」三段论
4. 引用数据或权威来源增加可信度（可适当虚构合理数据）
5. 结尾总结要点，提供「行动清单」

### 结构模板
```
## 前言（问题引入）
## 核心要点一
### 为什么重要
### 具体怎么做
### 真实案例
## 核心要点二（同上）
## 总结：行动清单
```

### 禁忌
- 不用口语化表达（「其实」「然后」「就是」）
- 不加无意义 emoji
- 不写未经验证的极端说法

## 风格 3：😊 轻松闲聊风

**适用场景**：生活日常、美食、旅行、情感类
**核心原则**：像和好友聊天，读起来轻松自然，有画面感

### 写作规则
1. 第一人称叙述，多用「我」「我们」
2. 大量口语词（「真的」「超级」「绝了」「OMG」「太爱了」）
3. 短句为主，每句不超过 20 字
4. 用括号插入内心独白（（我当时就傻了））
5. 配合场景描写，有细节有画面

### 示例
```
# 去了这家店，我已经回不去了😭

上周朋友拉着我去了一家巷子里的小馆子。

说实话，我刚进门的时候是有点懵的——
装修很破，就四张桌子，老板娘看起来不太好说话（苦笑）

但！一口下去——

（我沉默了大概三秒）

这是什么神仙味道？？？

蛋炒饭粒粒分明，火候刚刚好
汤是老板每天早上熬的，一点都不咸
价格还超便宜，人均30不到

我已经去了三次了
朋友说我有点上头，我说对，我就是上头了 😂

有没有人和我一样，总能在犄角旮旯找到宝藏小店？
```

## 风格 4：💪 励志鸡汤风

**适用场景**：情感、个人成长、逆袭故事类
**核心原则**：正能量、有共鸣、让人看完想转发

### 写作规则
1. 开头用反差或痛点勾住读者（「一年前我还在XXX，现在...」）
2. 中间讲故事，有起伏有转折
3. 适当插入金句（独立成段，加粗）
4. 结尾升华主题，给出人生感悟
5. 语气温暖真诚，避免说教感

### 金句模板
- **「不是你不行，是你还没找到对的方法。」**
- **「所有的厚积薄发，都经历过一段无人问津。」**
- **「别羡慕别人的结果，你没看到他们的过程。」**

## 风格 5：🎯 种草带货风

**适用场景**：商品推荐、好物分享、测评类
**核心原则**：真实体验感 + 利益点清晰 + 情感共鸣，促进转化

### 写作规则
1. 开头必须点出用户痛点（「找了半年终于找到了...」）
2. 产品介绍突出核心卖点（3个以内，不贪多）
3. 加入使用场景（「上班通勤」「晚上睡前」）
4. 真实感细节（价格、买了多少次、踩坑经历）
5. 结尾给出购买建议，降低决策门槛

### 结构
```
痛点引入 → 发现产品 → 核心卖点（2-3点）→ 使用体验 → 对比同类 → 购买建议
```

**适用场景**：准备拍摄抖音/快手视频时使用
**核心原则**：按镜头拆分，口播稿精炼，节奏快

```markdown
- **时长建议**：XX 秒
- **拍摄场景**：XXX
- **所需道具**：XXX

## 脚本

### 开头钩子（0-3秒）
**画面**：[镜头描述]
**口播**：「XXX」（语气：强烈/疑问/惊讶）

### 主体内容（3-X秒）
**画面**：[镜头描述]
**口播**：「XXX」

### 结尾 CTA（最后3秒）
**画面**：[镜头描述]
**口播**：「XXX，关注我，下期继续」

## 字幕重点
- 第X秒打出：「XXX」（大字幕）
- 第X秒打出：「XXX」
```
